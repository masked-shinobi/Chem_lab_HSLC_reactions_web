A premium, clean-coded chemistry reaction simulator main page. The UI features a central reaction vessel (flask), a bottom chemical collection shelf with draggable/clickable icons, and a floating info panel for reaction results. Use a glassmorphic aesthetic with deep gradients, smooth transitions, and high-quality typography. Include visual feedback like bubbles, precipitates, and color changes in the vessel.

**DESIGN SYSTEM (REQUIRED):**
- Utilize `glass` utility for all containers.
- Use `backdrop-blur-lg` and `border-white/10`.
- Headers should use `Space Grotesk` and `tracking-tight`.
- Floating action buttons should have a `glow` effect on hover.
- Background should be a deep radial gradient.

**Page Structure:**
1. Header with branding and "Lab Settings".
2. Main viewport with the central Flask.
3. Floating Right Panel: "Reaction Analysis" (Info Panel).
4. Bottom Shelf: "Chemical Collection" with searchable tiles.
